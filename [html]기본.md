# HTML

> 웹페이지를 만들기 위한 언어, 웹브라우저(크롬,파이어폭스) 위에서 동작

1. HTML 형식

   ```html
   <태그명 속성="속성값" 속성2="속성값2 속성값2-1">정보</태그명>
   ```

   ```html
   <span id="KOSPI_now" class="num num2">1,996.05</span>
   ```

2. HTML 문서의 구조

   ```html
   <!DOCTYPE html>
   <html>
   	<head>
           <meta charset="utf-8">
         	<title>웹페이지 제목</title>
   	</head>
   
   	<body>   
   	</body>
   </html>
   ```

   3. 태그

   - link : 문서에 필요한 다른 파일을 가져올 때 사용
   - h1 : 문서의 제목
   - a : 문서와 문서를 연결 주소
   - div : 여러가지 태그를 하나의 그룹으로 묶음
